## ⚡ What is VylexAI

VylexAI is a decentralized infrastructure that combines AI compute, distributed storage, and node-based resource sharing into a single unified network.

The platform enables:
- execution of AI tasks across distributed nodes  
- monetization of idle compute resources  
- scalable decentralized storage  

---

## 🔥 MVP Demonstration

This MVP demonstrates a working simulation of:

1. Client submits AI task  
2. Network scans available nodes  
3. Task is distributed across nodes  
4. Nodes process the task  
5. Result is returned to the client  

All processes are visualized in real time through the interface.

---

## 🧠 Multi-Role Architecture

VylexAI Core supports three key roles:

### 1. AI Clients
Users submit compute tasks (AI inference, data processing)

### 2. Node Providers
Participants contribute CPU/GPU resources and receive tasks

### 3. Storage Clients
Users store and retrieve data across distributed nodes
